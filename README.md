# upbh

Este repositório contém funções para extrair informações de textos semelhantes ao PDF do prontuário. 

## Executando os testes

1. Instale o Node.js (versão 16 ou superior).
2. Dentro do diretório do projeto, rode:

```bash
npm test
```

Isso executa `node tests/extraction.test.js` e imprime `Todos os testes passaram!` se tudo ocorrer bem.
